Pasos para compilar el proyecto

1. Conectar el suscriptor
	$ cd suscriptor
si quiere volver a compilar
	$ g++ -o suscriptor suscriptor.cpp -lpaho-mqtt3a
si quiere ejecutarlo
	$ ./suscriptor

2. Conectar el publicador
	$ cd publicador
si quiere volver a compilar
	$ g++ -o publicador publicador.cpp -lpaho-mqtt3a
si quiere ejecutarlo
	$ ./publicador
Una vez iniciado el publicador le pedira, host, puerto, topic y mensaje o payload
